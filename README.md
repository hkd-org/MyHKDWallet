### 如何在谷歌浏览器添加 GoHKD扩展（插件）

1.下载 [GoHKD](https://github.com/hkd-org/MyHKDWallet/releases/tag/master) 插件压缩包，并解压

2.检查使用的谷歌浏览器是否为最新版本

3.打开谷歌浏览器 > 设置（新版本的浏览器点击右上角的按钮）> 更多工具 > 扩展程序

4.把开发者选项打开

5.点击加载已解压程序，选择第一步解压得到的文件夹

6.确认添加，完成安装
